# TECH JOB ROLES ANALYSIS AND PREDICTIVE MODELLING WITH PYTHON
This repository investigates a dataset with over 250, 000 records. With information about about job titles, experience level, level of education, skills, certifications, industry, company size and location, my intention is to utlize power of Python in Data Science specifically for the divisions of Data Analysis and Predictive Modelling. This repository therefore comprises of two distinct folders: 
- The first folder is exclusively for Exploratory Data Analysis: To understand the data and uncover insights on how various factors affect Tech Job Roles in the 24th Century job Market.
- The Second Folder is exclusively for Predictive Modelling.
This entire project utilizes the Python  using Jupyter Notebooksconsists of two folders with Jupyter Notebooks

Job Salary Prediction Dataset (kaggle.com)
